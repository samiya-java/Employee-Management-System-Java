# Employee Management System (Java)

This is a console-based Employee Management System developed using Core Java as part of my internship at MainCrafts Technology.

## Project Description
The Employee Management System allows users to manage employee records through a menu-driven console application. Users can add, view, search, and update employee details.

## Features
- Add new employee
- View all employees
- Search employee by ID
- Update employee department
- Exit application

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner

## How to Run
1. Compile the program:
   javac Employee.java EmployeeApp.java
2. Run the program:
   java EmployeeApp
